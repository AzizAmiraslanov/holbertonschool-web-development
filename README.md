# holbertonschool-web-development

This repository contains projects for the Holberton School Web Development curriculum.
Each directory focuses on a specific topic, from HTML structure to advanced CSS styling and front-end development techniques.
Students build real-world web pages step by step while applying industry best practices and design specifications.
